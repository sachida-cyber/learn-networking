# Networking Mastery Tree Map

An interactive, offline learning website that visually explains computer networking concepts, protocols, and tools using D3.js.  
Built with HTML, CSS, and vanilla JavaScript. No frameworks, no backend, fully client-side.

---

## Project Overview

Networking Mastery Tree Map helps learners explore over 200 networking topics in a single visual structure.  
Each node represents a networking concept, tool, or command, explained in simple Hinglish (Hindi-English mix) for easy understanding.

This project was developed by SACHIDAX as an open-source knowledge map for students, professionals, and tech enthusiasts.

---

## Features

- Interactive D3.js collapsible tree with zoom and pan  
- Works fully offline (static HTML site)  
- Real-time search and topic filtering  
- Expand all / Collapse all controls  
- Light and dark theme toggle  
- Hinglish-based explanations for every concept  
- Right-side details panel with usage examples  
- Responsive layout for desktop and mobile  
- Learn Path section (Beginner → Intermediate → Advanced)

---

## Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript  
- D3.js v7 (via CDN)

---

## Structure

**index.html** — main structure and layout  
**style.css** — dark terminal-inspired design  
**script.js** — tree logic, data, and interactions  

All networking data is embedded as a JSON-like `const DATA = {}` inside `script.js`.

---

## How to Use

1. Download or clone the repository.  
2. Open `index.html` directly in your browser.  
3. Explore topics, search commands, and read explanations.  
4. Optionally, deploy it on GitHub Pages for public access.
